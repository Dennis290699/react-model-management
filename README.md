# La Angels Model Management

A luxury fashion modeling agency portfolio website featuring a dark/light mode, responsive design, and high-impact visual aesthetics.

## Developer

Developed by **Dennis290699**.

## Repository

Project Name: **react-model-management**

## Features

- **Modern UI/UX**: Built with React, Tailwind CSS, and Framer Motion.
- **Responsive Design**: Fully responsive layout for all devices.
- **Dark/Light Mode**: Seamless theme switching.
- **Model Portfolio**: Detailed model profiles with galleries.
- **Category Filtering**: Filter models by category (Women, Men, New Faces, etc.).

## Run Locally

**Prerequisites:** Node.js

1. Clone the repository:
   ```bash
   git clone https://github.com/Dennis290699/react-model-management.git
   cd react-model-management
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the app:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`.
